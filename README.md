# FinScript---Simulador-Avan-ado-de-Portf-lios-Financeiros
O FinScript é uma Linguagem de Domínio Específico (DSL) para simulações financeiras. Feito em Python/Flask , compila scripts e projeta rendimentos reais , aplicando impostos, aportes e indexadores do mercado brasileiro , com resultados exibidos em gráficos dinâmicos numa interface web.
